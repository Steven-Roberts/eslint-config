# ESLint config

The JavaScript lint configuration used for my projects

## Install
```shell
$ npm install --save-dev eslint Steven-Roberts/eslint-config
```

## Usage
Add the `extends` property to your ESLint config file

```json
{
    "extends": "steven-roberts",
    "rules": {
        "ruleToOverride": "..."
    }
}
```

## Test
```shell
$ npm test
```
